Label=(Pologne) Numéro identifiant TVA dans les transactions de taxe de vente
 ;Label for enable tax exempt number in sales tax transactions
Summary=Cette fonction contrôle la manière dont les numéros identifiant TVA sont renseignés dans les transactions de taxe de vente. Si elle est activée, le numéro identifiant TVA d’une transaction de taxe de vente est copié à partir de l’en-tête du document correspondant, par exemple commande vente, commande fournisseur ou facture financière. Sinon, le numéro identifiant TVA sera copié à partir des données principales client/fournisseur.
